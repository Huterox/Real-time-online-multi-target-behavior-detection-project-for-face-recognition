# Real-time online multi-target behavior detection project for face recognition

#### 介绍
集成Dlib实现人脸识别模块，以及通过YOLOV5+DeepSort+SlowFast 实现多目标实时在线行为检测。并且开发功能对接接口，可以快速进行二次开发。
